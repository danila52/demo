Инструкция по установке и запуску проекта
1. Установка зависимостей
Открой папку проекта в VS Code и выполни в двух терминалах:

# Терминал 1 — Backend
cd server
npm install

# Терминал 2 — Frontend
cd client
npm install

2. Запуск проекта

# Терминал 1 — Запуск сервера
cd server
npm run dev

# Терминал 2 — Запуск сайта
cd client
npm run dev

3. Открыть в браузере
Фронтенд: http://localhost:5173
